Welcome to my first blog post! Today I'm excited to share how I built this static blog using **Astro** and **MDX**.

## Why Astro and MDX?

Astro is a fantastic static site generator that allows you to build fast, content-focused websites. When combined with MDX, you get the power of:

- **Markdown simplicity** for writing content
- **JSX components** for interactive elements
- **Zero JS by default** for optimal performance
- **Component islands** for when you need interactivity

## Getting Started

Setting up an Astro project with MDX is surprisingly straightforward:

```bash
# Create a new Astro project
npm create astro@latest my-blog -- --template minimal

# Install MDX integration
npm install @astrojs/mdx

# Configure Astro
# Add mdx() to your astro.config.mjs integrations
```

## Writing in MDX

MDX allows you to write JSX directly in your Markdown files. This means you can:

1. Use all the Markdown syntax you know and love
2. Import and use React/Astro components
3. Create dynamic content with JavaScript expressions

> MDX is the perfect blend of simplicity and power for content creators who want more than basic Markdown.
